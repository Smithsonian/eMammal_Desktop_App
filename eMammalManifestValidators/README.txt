The WCS validator is for manifests with IDs at the image levels where the program generates the sequence

The eMammal validator is for the files that have IDs at the sequence level and sparse information at the image level

The manifests are named for the validator that should be used

To use the jar file open it and enter the file path to the folder contain the manifest in the place where it says "Input text here". You have to delete the example path. MAKE SURE to include / at the END

An example path is C:\Users\forrestert\Dropbox (Smithsonian)\eMammal Manifest Validators\Manifest Examples\
